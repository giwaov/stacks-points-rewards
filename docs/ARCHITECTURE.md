# Architecture

## Overview

On-chain loyalty points system on Stacks.

## Data Maps
- `balances`: principal to points balance
- `rewards`: reward ID to details and cost

## Functions
- `mint-points`: Owner mints points
- `transfer-points`: Transfer between users
- `redeem-reward`: Spend points on a reward
- `create-reward`: Owner creates a reward
- `get-balance`: Check points balance
